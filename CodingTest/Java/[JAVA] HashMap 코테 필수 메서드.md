# [JAVA] HashMap 코테 필수 메서드

JAVA 코테 풀때 필수로 알아야할 메서드를 정리하자.

| Modifier and Type | 메서드 | 설명 |
| --- | --- | --- |
| void | clear() | 모든 요소들을 삭제 |
| boolean | containsKey(Object key) | 특정 키가 HashMap에 존재한다면 true를 반환 |
| boolean | containsValue(Object value) | 특정 값이 HashMap에 존재한다면 true를 반환 |
| Set<Map.Entry<K, V>> | entrySet | HashMap에 있는 모든 쌍들을 Set형태로 반환 |
| V | get(Object key) | 키에 대한 값을 리턴, 없다면 Null 리턴 |
| V | getOrDefault(Obejct key, V defaultValue) | 키에대한 값을 리턴, 없다면 파라미터로 넘긴 defaultValue를 리턴 |
| V | put(K key, V value) | 키에 대한 값을 함께 HashMap에 삽입 |
| V | putIfAbsent(K key, V value) | HashMap에 키에 대한 값이 없다면 삽입후 null 리턴, 값이 있다면 값을 삽입하지 않고 기존 값을 리턴 |
| int | size() | HashMap이 가지고 있는 키-값 쌍들의 갯수를 반환 |
| V | remove(Object key) | key에 대한 값이 존재하면 해당 쌍을 HashMap에서 제거 |

### entrySet 사용 예시 (Generated By ChatGPT)

```java
Map<String, Integer> map = new HashMap<>();
map.put("apple", 3);
map.put("banana", 5);
map.put("orange", 2);

for (Map.Entry<String, Integer> entry : map.entrySet()) {
    System.out.println(entry.getKey() + " : " + entry.getValue());
}
```

apple : 3
banana : 5
orange : 2
